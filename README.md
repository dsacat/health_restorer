# Health Restorer 1.2.0

Готовый автономный Windows x64 EXE.

Исправлено автоматическое продолжение после Microsoft Defender Offline. Защищённый контроллер запускается при загрузке Windows и при входе пользователя, останавливает старое сломанное задание, ожидает готовности Task Scheduler, WMI и Microsoft Defender и повторяет запуск до 12 раз.

После Offline Scan обязательно выполняются:

1. полный Microsoft Defender Scan всех доступных файлов;
2. аудит и безопасное отключение сторонней автозагрузки;
3. проверка дисков, DISM и SFC;
4. очистка кэшей, временных файлов и корзины;
5. удаление подтверждённых остатков программ;
6. оптимизация накопителей;
7. очистка свободного пространства HDD или TRIM/ReTrim SSD.

Текущий этап:

```text
C:\Users\Public\Desktop\HealthRestorer-progress.txt
```

Журнал:

```text
C:\ProgramData\HealthRestorer\health-restorer.log
```

Коммерческое использование запрещено без письменного разрешения dsa_cat.
