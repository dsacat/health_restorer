# Health Restorer

`Health Restorer` — одноразовый сценарий глубокого обслуживания Windows 10/11. Он запрашивает права администратора, создаёт временные задания от имени `SYSTEM`, продолжает работу после перезагрузок и удаляет задания после завершения.

## Готовый EXE

```text
dist/HealthRestorer.exe
```

Контрольная сумма:

```text
dist/HealthRestorer.exe.sha256
```

EXE автономный и однофайловый. Отдельно устанавливать .NET и вручную размещать PowerShell-скрипты не требуется.

## Порядок работы

1. Обновляются базы Microsoft Defender.
2. Запускается Microsoft Defender Offline до обычной загрузки Windows.
3. После загрузки обязательно запускается отдельный полный Microsoft Defender Scan с ожиданием его завершения.
4. На время полной проверки программа пытается включить сканирование:
   - архивов;
   - почтовых хранилищ;
   - скриптов;
   - точек восстановления;
   - подключённых съёмных накопителей;
   - подключённых сетевых дисков и доступных сетевых файлов.
5. Пользовательские исключения Defender для путей, расширений и процессов временно снимаются и после проверки восстанавливаются. Если защиту настроек блокирует Tamper Protection или корпоративная политика, ошибка записывается в журнал.
6. Создаётся резервная копия и отключается сторонняя автозагрузка.
7. Выполняется предзагрузочная проверка системного диска.
8. DISM восстанавливает хранилище компонентов Windows.
9. SFC проверяет и восстанавливает защищённые системные файлы.
10. `CHKDSK /scan` проверяет все локальные диски.
11. Очищаются временные файлы, кэш Windows Update, отчёты об ошибках, пользовательский кэш, кэши браузеров, шейдерный кэш, DNS-кэш, эскизы и корзина.
12. Удаляются подтверждённые остатки не до конца удалённых программ:
    - сломанные записи деинсталляции;
    - сторонние службы и задания с отсутствующими файлами;
    - битые ярлыки;
    - пустые и подтверждённо оставшиеся каталоги программ.
13. Локальные накопители оптимизируются через `defrag /O`, который выбирает подходящую операцию для HDD или SSD.
14. После основного обслуживания очищаются остатки ранее удалённых файлов:
    - на HDD свободное пространство перезаписывается через `cipher /w`;
    - на SSD выполняется `TRIM/ReTrim`.
15. Создаются журнал и итоговый отчёт.

## Что означает «полная проверка всех файлов»

Microsoft Defender Full Scan проверяет все доступные файлы на смонтированных локальных дисках и, при включённых параметрах, на подключённых съёмных и сетевых накопителях. Запуск выполняется через:

```text
MpCmdRun.exe -Scan -ScanType 2 -Timeout 30
```

Программа ждёт завершения процесса сканирования. Стандартный максимальный срок проверки увеличен до 30 дней.

Невозможно проверить данные, которые физически недоступны операционной системе: отключённые диски, не загруженные локально облачные файлы, повреждённые нечитаемые области, содержимое некоторых зашифрованных контейнеров и архивов с неизвестным паролем.

Отчёт полной проверки сохраняется в:

```text
C:\ProgramData\HealthRestorer\defender-full-scan-report.txt
```

Настройки Defender до проверки сохраняются в:

```text
C:\ProgramData\HealthRestorer\defender-preferences-before-full-scan.json
```

## Защита при очистке программных остатков

Сценарий не удаляет произвольную папку только из-за неизвестного названия. Остаток удаляется лишь при сочетании нескольких признаков: отсутствует деинсталлятор, запись не относится к Windows Installer или Microsoft, а в каталоге отсутствуют работающие исполняемые компоненты.

Перед изменением сохраняются:

- экспорт разделов реестра;
- XML заданий Планировщика;
- разделы реестра служб;
- копии битых ярлыков;
- JSON-отчёт.

Резервная копия метаданных:

```text
C:\ProgramData\HealthRestorer\ProgramResidueBackup-ГГГГММДД-ЧЧММСС
```

## Запуск

1. Сохрани открытые файлы.
2. Создай резервную копию важных данных.
3. Подключи компьютер к питанию.
4. Запусти `HealthRestorer.exe`.
5. Подтверди запрос прав администратора.

Компьютер перезагрузится несколько раз. Полная проверка и очистка свободного пространства могут продолжаться много часов.

## Файлы проекта

- `HealthRestorer.ps1` — Offline Scan, обязательный Full Scan, DISM, SFC, диски, кэш и оптимизация.
- `ProgramResidueCleanup.ps1` — консервативная очистка остатков удалённых программ.
- `SecureDeletedData.ps1` — очистка программных остатков и свободного пространства.
- `Start-HealthRestorer.ps1` — единая точка запуска и регистрация завершающего задания.
- `src/HealthRestorer.Launcher/` — исходный код EXE-запускателя.
- `.github/workflows/build-exe.yml` — проверка синтаксиса, обязательных этапов, метаданных и сборка EXE на Windows runner.

## Ограничения

- Для антивирусных этапов требуется активный Microsoft Defender.
- Tamper Protection или корпоративная политика могут запретить временное изменение отдельных настроек и исключений; такие случаи фиксируются в журнале.
- DISM может использовать Windows Update.
- Открытые системой файлы кэша иногда невозможно удалить до следующей перезагрузки.
- На SSD обычная программа не может гарантировать физическую перезапись каждой ячейки из-за контроллера и выравнивания износа. Для полного уничтожения всего SSD нужен Secure Erase/Sanitize всего накопителя.
- Сценарий не заменяет резервное копирование и не исправляет физически неисправный накопитель.
