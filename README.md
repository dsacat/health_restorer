# Health Restorer 1.2.0

Готовая релизная сборка для Windows 10/11 x64.

## Файлы

- `HealthRestorer.exe` — автономный исполняемый файл.
- `HealthRestorer.exe.sha256` — контрольная сумма SHA-256.
- `LICENSE.md` — некоммерческая лицензия.

## Что выполняется

1. Microsoft Defender Offline до обычной загрузки Windows.
2. Обязательная полная проверка Microsoft Defender всех доступных файлов с ожиданием завершения.
3. Проверка и восстановление Windows через DISM и SFC.
4. Проверка и оптимизация локальных дисков.
5. Очистка системного, пользовательского и браузерного кэша, временных файлов и корзины.
6. Очистка подтверждённых остатков удалённых программ.
7. Очистка свободного места на HDD или TRIM/ReTrim на SSD.

Полная проверка включает подключённые локальные диски, подключённые съёмные накопители, архивы, почтовые хранилища, точки восстановления и доступные сетевые файлы. Исключения Defender временно снимаются и восстанавливаются после проверки.

Недоступные зашифрованные данные, защищённые паролем архивы, отключённые носители и не загруженные локально облачные файлы проверить невозможно.

## Запуск

1. Сохрани важные файлы и закрой программы.
2. Подключи компьютер к питанию.
3. Запусти `HealthRestorer.exe`.
4. Подтверди права администратора.

Компьютер перезагрузится несколько раз. Полная проверка может продолжаться много часов.
