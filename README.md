# Health Restorer 1.2.0

Готовая релизная сборка для Windows 10/11 x64.

## Файлы

- `HealthRestorer.exe` — автономный исполняемый файл.
- `HealthRestorer.exe.sha256` — контрольная сумма SHA-256.
- `LICENSE.md` — некоммерческая лицензия.

## Запуск

1. Сохрани важные файлы и закрой программы.
2. Подключи ноутбук к питанию.
3. Запусти `HealthRestorer.exe`.
4. Подтверди запрос прав администратора.

Компьютер перезагрузится несколько раз. Не выключай его во время проверки и восстановления.
